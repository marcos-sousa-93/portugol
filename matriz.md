# Declaração de MATRIZ
```Portugol
  // Declaração de uma matriz de números reais com 5 linhas e 3 colunas
  real matriz[5][3]

  // Gravar um valor na matriz na posição 0 (primeira linha) e 1 (segunda coluna)
  matriz[0][1] = 2.5

  // Mostrar o valor na matriz
  escreva(matriz[0][1])
```

|-------|-------|-------|-------|-------|-------|

|   1   |       |       |       |       |       |
|   2   |       |       |       |       |       |
