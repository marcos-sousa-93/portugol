# Operações lógica NÃO
```Portugol
// Nesse caso teste a variável teste foi inicializada como falso, e foi verificado se teste não é verdadeiro
logico teste = falso

se(nao(teste))
{
  escreva("Teste positivo")
}

// nesse caso teste a soma das variáveis A e B resulta em 5, e comparado se a mesma é maior que 7
// entretanto o operador NAO, verifica se A + B não são maiores que 7
inteiro a = 2, b = 3

se(nao(a + b > 7)
{
  escreva("Teste positivo")
}
```
