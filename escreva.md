# Escreva (SAÍDA)
```Portugol
inteiro num = 5

// Escreve no console um texto qualquer
escreva("Escreva um texto aqui.\n")

// Escreva no console o valor da variável "num"
escreva(num, "\n")

// Escreva no console o resultado da operação
escreva(num + num, "\n")

// Escreva no console o texto digitado e o valor contido na variavel
escreva("O valor da variável é ", num)

// Escreva no console o texo com quebra de linha
escreva("Texto com\n","quebra-linha")

// Escreva no console o texto com espaço de tabulação
escreva("Texto com\t tabulação")
```
